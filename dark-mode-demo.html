⑧ 深色 vs 浅色 — 设计系统全对比
🌑 深色版
背景：#0a0a0a
文字：#f0f0f0 / .45 / .28
头像：圆形，写实连环画人像
身份：彩色发光（glow emission）
发言：光晕脉冲 + 涟漪圈
情绪：电影感 · 权力感 · 紧张
场景：夜间 / 高风险决策 / 展示
🌤 浅色版
背景：#f4f3ef
文字：#1a1a1a / .55 / .3
头像：圆角方形，朱砂印章
身份：彩色边框（border identity）
发言：方形波纹 + 投影扩散
情绪：商务感 · 理性 · 人文东方
场景：晨间 / 展示给评委 / 日常
切换方案
两套共用同一套25色身份系统 → 切换时有认知连续性。
实现：:root[data-theme='light'] 全局覆盖核心CSS变量，约50行。
建议：黑巅现场演示用深色（震撼），对外展示/日常用浅色（亲和）。
