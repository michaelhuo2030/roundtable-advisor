# 参谋 · 头像颜色系统设计文档
## Avatar Color Identity System — Design Guide

> **预览文件：** `avatar-color-demo.html`（在浏览器直接打开）

---

## 核心原则

- **黑色主题**：背景 `#0a0a0a`，不用白色/浅色版
- **颜色即身份**：每位顾问有且仅有一个专属颜色，颜色是其"签名"
- **连环画风格**：头像来自白描/连环画 AI 生成图，黑白线条 + CSS 调色
- **三种状态**：默认 → 激活（彩环）→ 发言中（脉动动画）

## 头像裁剪技术

源图为 5×5 网格，25人全在一张图里。用 CSS 裁剪，无需切图：

```css
.av-inner {

  background-size: 500% 500%;       /* 5列×5行 */
  background-position: 25% 0%;     /* 第2列第1行 = 乔布斯 */
  filter: brightness(1.15) contrast(1.3) grayscale(0.1);
}
```

## CSS 实现结构

```html
<!-- 每个头像的 HTML 结构 -->
<div class="av md lit" data-p="jobs">
  <div class="av-inner p-jobs"></div>
</div>
```

```css
/* CSS 变量绑定颜色 */
[data-p="jobs"] {
  --c: #B0B0FF;
  --c-glow: rgba(176,176,255,.5);
  --c-glow2: rgba(176,176,255,.15);
}

/* 状态类 */
.av          /* 默认：细白边框，无光效 */
.av.lit      /* 激活：彩色边框 + 柔光 */
.av.speaking /* 发言中：彩色脉动 + ripple扩散环 */
.av.dim      /* 暗淡：灰化低透明度 */
```
